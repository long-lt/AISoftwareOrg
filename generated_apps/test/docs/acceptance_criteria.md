# Acceptance Criteria: Test

## Tiêu Chí Chung

- App build được trên target platform đã chọn.
- Navigation chính hoạt động không lỗi.
- UI tuân thủ style `modern`.
- Không có lỗi null-safety trong flow MVP.

## Tiêu Chí Theo Tính Năng

### Todo

- Given người dùng mở app, when truy cập todo, then màn hình hoặc chức năng tương ứng hiển thị đúng.
- Given dữ liệu rỗng hoặc lỗi, when todo được tải, then app hiển thị empty/error state rõ ràng.
- Given thao tác hợp lệ, when người dùng hoàn tất hành động trong todo, then trạng thái UI được cập nhật đúng.
