# Screen List: Test

| Screen | Mục đích | Entry | State cần hỗ trợ |
| --- | --- | --- | --- |
| HomeScreen | Tổng quan app, entry point chính | User mở app | Điều hướng tới feature chính |
| TodoScreen | Màn hình xử lý Todo | User chọn Todo | Hiển thị loading, empty, success, failure |
| DashboardScreen | Màn hình xử lý Dashboard | User chọn Dashboard | Hiển thị loading, empty, success, failure |
| SettingsScreen | Màn hình xử lý Settings | User chọn Settings | Hiển thị loading, empty, success, failure |
| SettingsScreen | Tuỳ chỉnh cơ bản và thông tin app | User mở settings | Hiển thị cấu hình và trạng thái app |

## Feature Mapping

- Todo -> TodoScreen
- Dashboard -> DashboardScreen
- Settings -> SettingsScreen
