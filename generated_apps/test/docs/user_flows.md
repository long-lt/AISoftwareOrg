# User Flows: Test

## Primary Persona

- Customer: người dùng chính cần hoàn thành tác vụ nhanh trên mobile.

## Flows

### Todo Flow

1. Customer mở app và vào module Todo.
2. App tải dữ liệu Todo từ repository/data source.
3. Customer xem danh sách hoặc nội dung chi tiết.
4. Customer thực hiện hành động chính: create_task, update_task, complete_task, filter_tasks.
5. App cập nhật UI state hoặc hiển thị lỗi rõ ràng nếu thao tác thất bại.
