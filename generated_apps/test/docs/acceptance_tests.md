# Acceptance Tests: Test

Các test này là contract để QA/DEV Agent sinh test tự động ở các phase sau.

## Test Cases

### Todo

- [ ] Load success: màn hình Todo hiển thị dữ liệu chính.
- [ ] Empty state: màn hình Todo hiển thị empty state khi không có dữ liệu.
- [ ] Error state: màn hình Todo hiển thị retry khi data source lỗi.
- [ ] Main action: thao tác chính của Todo cập nhật trạng thái UI đúng.
